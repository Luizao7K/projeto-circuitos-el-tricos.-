Projeto de Monitoramento de Temperatura e Umidade com IoT

Este projeto acadêmico foca na criação de um sistema de monitoramento ambiental utilizando um sensor DHT11 para coletar dados de temperatura e umidade. A arquitetura do sistema é composta por um microcontrolador Arduino Uno que lê os dados do sensor e os transmite para um módulo ESP32, que funciona como o gateway IoT, enviando as informações para a nuvem.

Componentes do Projeto

  • Arduino Uno
  • Módulo ESP32 (como módulo de comunicação Wi-Fi)
  • Sensor de temperatura e umidade DHT11
  • Protoboard
  • Cabos Jumper
  • Resistor de 4.7kΩ (necessário para a leitura do DHT11)

    Código-Fonte para o Arduino Uno

    O código a seguir foi desenvolvido no Arduino IDE e é responsável por inicializar o sensor, realizar as leituras periódicas de temperatura e umidade, e imprimir os resultados no monitor serial para serem capturados pelo módulo ESP32. As leituras são formatadas para facilitar a comunicação entre os dois microcontroladores.

    
#include <DHT.h>

#define DHTPIN 2      // Pino digital onde o sensor DHT11 está conectado
#define DHTTYPE DHT11   // Tipo do sensor: DHT11

// Inicializa o sensor DHT
DHT dht(DHTPIN, DHTTYPE);

void setup() {
  Serial.begin(9600); // Inicia a comunicação serial a 9600 bps
  Serial.println("Iniciando o sensor DHT11...");
  dht.begin();
}

void loop() {
  // A leitura do sensor DHT11 leva cerca de 250ms,
  // então é bom ter um intervalo de pelo menos 2 segundos entre leituras.
  delay(2000);

  // Lê a umidade e a temperatura em graus Celsius
  float h = dht.readHumidity();
  float t = dht.readTemperature();

  // Verifica se a leitura falhou (retorna 'NaN')
  if (isnan(h) || isnan(t)) {
    Serial.println("Erro: Falha ao ler do sensor DHT11!");
    return;
  }

  // Envia os dados pela porta serial em um formato legível para o ESP32
  // Exemplo de saída: "T:25.50,H:60.20"
  Serial.print("T:");
  Serial.print(t);
  Serial.print(",H:");
  Serial.println(h);
}
